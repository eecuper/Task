第四步：条件设置

安全优化

1.任务发布间隔：

每隔    发布 任务 {$sysConfig['num_intval_db']} 0 D币

选择此项服务后，会将您的任务分批发布，以避免订单过于集中：为了安全性建议选择此项服务

2.延长买家购物周期： 仅推荐 重复购买率低的商品使用，如家居，家电，高单价的商品等；其他品类不推荐使用

选择此项服务后，购买过任务商品的买家 1个月 内，将不能再接手包含本任务商品的任务；

3.任务佣金消费：

选择此项服；该功能是提升订单任务效率的设置，消费越高，系统将提升订单的优先级，进行任务执行操作

合计规则：(组合币 + 间隔币 + 佣金消费) X 订单数=总支付

合计收费： ( {$tk_db_more} + {$sysConfig['num_intval_db']} 0 + {$tk['db_yongjin']} ) X {$tk.taocan_number} 单 = {$tk.db_pay} D币；

4.押金支付：

合计规则：单价 X 数量 X 订单数+邮费 X 订单数=总押金

合计押金： {$tk.price} 元 X {$tk.cnt} 件 X {$tk.taocan_number} 单 + {$tk['transport']} X {$tk['taocan_number']} = {$tk[price]*$tk['cnt']*$tk['taocan_number']+$tk['transport']*$tk['taocan_number']} 元；

优质好评 将有助于提升评价质量并优化您商品评价映像关键词
